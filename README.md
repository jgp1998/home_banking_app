#Home Banking App
##Cliente: Banco Saint Patrick

##Contexto: La mayoría de los bancos ofrecen plataformas digitales, pero muchas son complejas o inseguras. Este proyecto busca crear una experiencia sencilla y confiable para que los usuarios gestionen su dinero fácilmente.

##Objetivo: Permitir que los usuarios realicen transferencias entre cuentas de forma rápida y segura, asegurando que los datos estén protegidos y las transacciones validadas correctamente.
